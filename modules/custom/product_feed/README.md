# Product Feed

Модуль `product_feed` додає маршрут `/product.xml`, який повертає XML-фід з публікованих нод типу `product`.

## Особливості

- Формує абсолютні посилання, використовуючи домен, з якого прийшов запит. Таким чином на кожному мовному домені посилання ведуть на відповідний домен.
- Враховує мовні переклади: для кожної ноди використовується переклад поточної мови домену, або ж дефолтної мови сайту.
- Видає лише опубліковані матеріали та кешується на 15 хвилин.

## Встановлення

1. Розмістіть модуль у директорії `modules/custom` вашого проєкту Drupal.
2. Активуйте модуль командою `drush en product_feed` або через UI.
3. Переконайтесь, що ноди типу `product` мають поля `field_sku` та `field_price`, або адаптуйте код сервісу до ваших полів.

Після встановлення маршрут `/product.xml` стане доступним на кожному домені.
